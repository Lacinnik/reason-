# Privacy / Конфиденциальность

Resonant Translation Engine выполняет перевод в браузере пользователя.

- Введённый текст не отправляется на сервер проекта.
- Нейромодели и Transformers.js загружаются из публичных источников только для первичной подготовки и затем используются из браузерного кэша.
- Переводная память, словарь и настройки сохраняются локально в IndexedDB/localStorage текущего браузера.
- Телеметрия, рекламные идентификаторы и аналитические счётчики отсутствуют.
- Экспорт памяти создаётся только по явному нажатию пользователя.
- Очистка данных сайта в браузере удаляет локальную память RTE.

GitHub Pages обслуживает статические файлы приложения; GitHub может вести собственные стандартные журналы HTTP-доступа в соответствии со своей политикой.
