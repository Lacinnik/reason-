# Resonant Translation Engine — RTE v1

Браузерный EN ↔ RU переводчик на Transformers.js.

## Публикация на GitHub Pages

1. Создайте публичный репозиторий, например `rte`.
2. Загрузите в корень все файлы из этого архива.
3. GitHub → Settings → Pages.
4. Source: `Deploy from a branch`.
5. Branch: `main`, folder: `/ (root)`.
6. Сохраните. Адрес будет вида `https://lacinnik.github.io/rte/`.

## Первый запуск

Первый перевод требует интернет: браузер скачивает Transformers.js и модель выбранного направления. Нажмите «Подготовить офлайн» для EN→RU, затем переключите направление и повторите для RU→EN.

## Важные ограничения

- Модели указаны как `Xenova/opus-mt-en-ru` и `Xenova/opus-mt-ru-en`. Если конкретный репозиторий модели станет недоступен или несовместим с текущей версией Transformers.js, идентификатор можно заменить прямо в блоке «Диагностика и модели».
- GitHub Pages должен обслуживать приложение по HTTPS; прямое открытие `index.html` как локального файла не позволит Service Worker работать.
- На iPhone модель может выгружаться системой при нехватке памяти или очистке данных Safari.
- Метрики α/Q/Cm/T в v1 являются эвристической локальной оценкой результата, а не отдельной моделью контроля качества.
