# Resonant Translation Engine — RTE v2.0.0

Локальный браузерный переводчик EN ↔ RU на Transformers.js. После первой подготовки моделей перевод работает без сети в пределах сохранности браузерного кэша.

**Рабочий адрес:** `https://lacinnik.github.io/reason-/`

## Лаборатория РЕЗОН

- **Translator 2.0 stable:** корневой автономный EN ↔ RU контур.
- **Collective Field Check 1.0 release candidate:** `https://lacinnik.github.io/reason-/field-check/` — локальная проверка предъявленности связей и порога проводимости.
- **7 Transmissions 1.0 stable:** `https://lacinnik.github.io/reason-/transmissions/` — самостоятельная колода семи режимов перехода с явными воротами голосов и локальным паспортом сессии.

Все три публичных входа обслуживаются одним Service Worker. Field Check и 7 Передач входят в предварительный кэш и регистрируют его при прямом открытии своего URL.

## Что входит в v2.0.0

RTE теперь выполняет не одиночный вызов модели, а полный контур:

`текст → сегментация → память → словарь → кандидаты → обратная мера → форма`

- Длинный текст делится на смысловые сегменты и собирается обратно с сохранением абзацев.
- Переводная память хранит утверждённые пары локально в IndexedDB.
- Результат можно исправить и нажать **«Принять в память»**.
- Словарь удерживает утверждённые EN ↔ RU термины.
- Режим трёх кандидатов формирует альтернативы и ранжирует их.
- **Глубокая проверка** переводит вариант обратно и повышает достоверность α.
- WebGPU используется только при доступности; при ошибке движок автоматически возвращается к WASM q8.
- В памяти удерживается только один живой нейронный pipeline; при смене направления предыдущий освобождается, а его веса остаются в браузерном кэше.
- Панель офлайн-диагностики показывает готовность обоих направлений и состояние хранилища.
- Память, словарь и настройки экспортируются одним JSON-пакетом.

## Первый запуск и офлайн

1. Откройте приложение по HTTPS.
2. Раскройте **«Офлайн-контур и устойчивость»**.
3. Нажмите **«Подготовить обе модели»** и дождитесь завершения.
4. При возможности нажмите **«Защитить хранилище»**.
5. Отключите сеть и выполните контрольный перевод в обе стороны.
6. На iPhone добавьте страницу на экран «Домой».

Статус «готово» фиксирует успешный запуск модели в текущем браузере. iOS всё равно может очистить кэш при критическом дефиците места.

## Метрики

- **α** — соответствие; при глубокой проверке основано на близости исходника и обратного перевода.
- **Q** — агрегированное качество формы.
- **Cₘ** — сохранение чисел, версий, URL, email, аббревиатур и утверждённых терминов. Эти инварианты защищаются до запуска модели и восстанавливаются после перевода.
- **T** — текучесть: длина, пунктуационная завершённость и отсутствие повторов.

Это локальная инженерная оценка, а не внешняя экспертная аттестация перевода.

## Техническая структура

- `index.html` — семантическая разметка интерфейса PWA.
- `styles.css` — адаптивный интерфейс для iPhone и настольных браузеров.
- `app.js` — оркестрация моделей и интерфейса.
- `engine-core.js` — сегментация, словарь, метрики, ранжирование.
- `storage.js` — IndexedDB и резервный localStorage.
- `vendor/transformers-3.7.2.js` — локальный загрузчик строго зафиксированной версии 3.7.2; Service Worker кэширует загрузчик и внешний ESM-рантайм после первого доступа.
- `sw.js` — кэш оболочки.
- `tests/` — 28 тестов RTE и 6 проверок опубликованных вертикалей РЕЗОН; 34 теста в полном прогоне.
- `PRIVACY.md` — границы локальной обработки и хранения данных.

## Автоматическая проверка перед публикацией

```bash
npm run check
```

Команда выполняет синтаксическую проверку и 34 модульных и статических теста. В Pull Request дополнительно запускается GitHub Actions quality gate и HTTP-smoke test. Полная нейронная инференция всё равно проверяется в браузере, поскольку модели исполняются через WebAssembly/WebGPU.

## Ограничения

- Модели: `Xenova/opus-mt-en-ru` и `Xenova/opus-mt-ru-en`.
- Первый запуск каждого направления требует интернет и свободное место.
- Глубокая проверка загружает обратную модель и требует заметно больше времени.
- Браузер может удалить данные сайта; регулярно экспортируйте пакет памяти.
