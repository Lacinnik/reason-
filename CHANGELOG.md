# Changelog

## 2.0.0-rc.1 — Resonant Translation Engine · clean rebuild

- Чистая пересборка без временных chunk/workflow-артефактов.
- Смысловая сегментация длинных текстов с обратной сборкой абзацев.
- Три кандидата перевода и локальное ранжирование по α/Q/Cₘ/T.
- Опциональная глубокая проверка через обратный перевод.
- Редактируемый результат и утверждение пользовательской правки.
- Локальная переводная память в IndexedDB с быстрыми точными совпадениями.
- Двунаправленный терминологический словарь и активная защита чисел, версий, URL, email и аббревиатур.
- Экспорт и импорт памяти, словаря и настроек одним JSON-пакетом.
- Автовыбор WebGPU с безопасным fallback на WASM q8.
- Явное освобождение нейронного pipeline при смене направления для снижения пикового потребления памяти.
- Панель состояния офлайн-моделей, оболочки, квоты и устойчивости хранилища.
- Строго зафиксированный Transformers.js 3.7.2 через локальный загрузчик; модуль, WASM и модели кэшируются браузером.
- Новый PWA-интерфейс, иконка, обновлённый Service Worker, 26 тестов и GitHub Actions quality gate.

## 0.1.0 — Foundation

- Двунаправленный EN ↔ RU перевод.
- Кэширование моделей и базовый офлайн-режим.
- Первая версия метрик α/Q/Cₘ/T.
