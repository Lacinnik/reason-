RTE v2.0.0-rc.1 — ЧИСТАЯ ПУБЛИКАЦИЯ НА WINDOWS

1. Полностью распакуйте ZIP в обычную папку.
2. Дважды нажмите PUBLISH_RTE_V2.cmd.
3. Разрешите установку Git/GitHub CLI, если Windows предложит.
4. Завершите вход в GitHub в открывшемся браузере.

Скрипт сам:
- пересоздаст feature/rte-v2 строго от актуальной main;
- удалит из ветки старые chunk/base64 и assembly-workflow;
- прогонит локальные тесты, если установлен Node.js;
- загрузит чистый RTE v2;
- закроет старый PR #1 как заменённый;
- создаст новый Draft Pull Request.

Рабочая версия сайта в main не заменяется этим запуском.
Не сливайте новый PR, пока GitHub Actions не станет зелёным и пока не выполнена проверка на iPhone из TESTING.md.
