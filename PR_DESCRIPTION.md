## RTE v2.0.0-rc.1 — clean release candidate

Этот PR заменяет неудачную ветку v0.2 чистой производственной сборкой без chunk/base64-артефактов и временных assembly-workflow.

### Движок

- смысловая сегментация длинного текста;
- точная и приближённая локальная переводная память;
- редактирование результата и принятие пользовательской коррекции;
- двунаправленный терминологический словарь;
- защита чисел, версий, URL, email и аббревиатур;
- один или три кандидата с резонансным ранжированием;
- глубокая проверка через обратный перевод;
- метрики α / Q / Cₘ / T;
- WebGPU с безопасным возвратом к CPU/WASM q8;
- освобождение нейронного pipeline при смене направления для устойчивости на iPhone.

### Офлайн и PWA

- зафиксирован Transformers.js 3.7.2;
- Service Worker кэширует оболочку, runtime и WASM-компоненты;
- модели остаются в браузерном кэше Transformers.js;
- диагностика готовности обоих направлений, квоты и persistence;
- импорт/экспорт памяти, словаря и настроек.

### Проверки

- `npm ci --ignore-scripts`;
- `npm run check`;
- 26/26 unit/static/storage tests;
- HTTP smoke test всех runtime-файлов;
- GitHub Actions `Verify RTE v2`;
- Pages deployment зависит от успешной проверки.

### Приёмка до merge

Не сливать до выполнения `TESTING.md` на iPhone и компьютере: EN→RU, RU→EN, онлайн, офлайн, длинный текст, память, словарь, кандидаты и глубокая проверка.
