# RTE v2.0.0 — протокол регрессионной приёмки

**Статус релиза:** пройдено перед выпуском `v2.0.0`.

Матрица обязательна для каждого следующего релиза. WebGPU — необязательное ускорение; успешный WASM q8 является нормой.

## Среда

- Windows 11: актуальный Chrome **или** Edge.
- iPhone: актуальный Safari, затем приложение, добавленное на экран «Домой».
- Для первого запуска — устойчивый Wi‑Fi и свободное место в браузере.

## Последовательность проверки

| ID | Действие | Ожидаемый результат |
|---|---|---|
| B1 | В Windows Chrome или Edge перевести оба коротких текста. | Оба направления работают; интерфейс не зависает; активен WASM q8 либо WebGPU с возвратом к WASM. |
| B2 | Перевести оба текста с инвариантами. | `v2.0.0`, URL, `qa@example.org`, `API` и `42` сохранены дословно. |
| B3 | Переключить направление. | Текущий перевод переносится в поле исходника; языки меняются местами. |
| B4 | В Windows запустить перевод через `Ctrl+Enter`. | Результат эквивалентен запуску кнопкой. |
| R1 | Выбрать `3 · резонанс`, перевести короткий текст, выбрать другую карточку при её наличии. | Нет ошибки; отображается от одной до трёх уникальных форм; выбор меняет основной результат и метрики. |
| R2 | Включить обратный перевод и перевести короткий текст. | Загружается обратная модель; результат остаётся читаемым; α пересчитывается. |
| M1 | Перевести `Memory source`, заменить результат на `Memory correction`, нажать «Принять в память», затем повторить тот же исходник. | Точный повтор возвращает `Поле удерживает меру.` без нового запуска модели. |
| M2 | Добавить `axis ↔ ось`; перевести `The axis keeps form.`, затем `Ось сохраняет форму.` | В первом результате есть `ось`, во втором — `axis`. |
| M3 | Экспортировать пакет, очистить память, импортировать пакет. | Утверждённая запись и словарь восстановлены. |
| L1 | Перевести `Long text`. | Три абзаца не обрезаны и не переставлены; инварианты сохранены. |
| O1 | Нажать «Подготовить обе модели», дождаться готовности EN→RU и RU→EN, перезагрузить страницу. | Оболочка PWA и обе модели отображают готовность. |
| O2 | Включить авиарежим и перевести новые, ранее не использованные фразы в обоих направлениях. | Оба перевода выполняются без сети, а не только из памяти. |
| I1 | На iPhone Safari выполнить B1–B3, M1–M2, L1, O1–O2. | Все обязательные сценарии проходят. |
| I2 | Safari → «Поделиться» → «На экран “Домой”»; открыть PWA и повторить O2. | PWA работает в standalone-режиме и переводит офлайн. |

## Опубликованные вертикали РЕЗОН

| ID | Действие | Ожидаемый результат |
|---|---|---|
| F1 | Открыть `/field-check/` прямой ссылкой в настольном браузере и Safari iPhone, собрать два голоса и провести проверку. | Страница запускается без внешних runtime-зависимостей; ALLOW, HOLD по связям и HOLD по метрикам формируются согласно воротам. |
| F2 | После результата Field Check нажать «Экспорт JSON», открыть файл и перезапустить страницу. | JSON содержит схему `reson.collective-meta-decision/1.0.0`, голоса, метрики и gate; последнее решение остаётся в localStorage. |
| T1 | Открыть `/transmissions/` прямой ссылкой, назвать объект, напряжение и два голоса, выбрать Передачу и зафиксировать след. | Формируется READY/HOLD-паспорт схемы `architectonica.seven-transmissions-session/1.0.0`; данные остаются в браузере. |
| T2 | После первого открытия Field Check и 7 Передач отключить сеть и перезапустить оба адреса. | Общий Service Worker отдаёт оболочки и локальные runtime-файлы обеих вертикалей из кэша. |

На iPhone недоступность кнопки «Защитить хранилище» — допустимое ограничение iOS. Потеря моделей или памяти после штатной очистки данных Safari не считается дефектом.


- **blocker** — не работает перевод или обязательный офлайн-контур.
- **major** — теряются инварианты, память, словарь или PWA-режим.
- **minor** — визуальная проблема без потери функции.
