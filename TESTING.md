# RTE v2.0.0-rc.1 — приёмочная матрица

## 1. Базовый перевод

- EN → RU: короткое предложение, абзац, текст с числами и URL.
- RU → EN: короткое предложение, абзац, текст с email и аббревиатурой.
- Переключение направления переносит текущий перевод в поле исходника.
- `Cmd/Ctrl + Enter` запускает перевод.

## 2. Резонансные формы

- Режим `1 · быстро` возвращает одну форму.
- Режим `3 · резонанс` показывает альтернативы, если модель поддерживает несколько последовательностей.
- Выбор карточки кандидата меняет основной результат и метрики.
- Глубокая проверка загружает обратную модель и пересчитывает α.

## 3. Память и словарь

- После `Принять в память` повторный точный запрос возвращается без инференции.
- Редактирование результата перед принятием сохраняет пользовательскую форму.
- Новый термин EN ↔ RU применяется в обоих направлениях.
- Экспортированный JSON импортируется после очистки памяти.

## 4. Длинный текст

- Текст из нескольких абзацев переводится без обрезания.
- Абзацы сохраняют порядок и разделение.
- Числа, версии, URL, email и аббревиатуры сохраняются дословно.

## 5. Офлайн

1. Онлайн нажать `Подготовить обе модели`.
2. Дождаться успешного запуска каждого направления.
3. Перезагрузить приложение один раз, чтобы Service Worker контролировал страницу.
4. Включить авиарежим.
5. Повторить EN → RU и RU → EN.
6. Проверить точное совпадение из памяти без модели.

## 6. Устройства

- iPhone Safari / добавленная на экран «Домой» PWA.
- Windows: Chrome или Edge.
- Проверить WASM-контур; WebGPU считать дополнительным ускорением, а не обязательным условием.
